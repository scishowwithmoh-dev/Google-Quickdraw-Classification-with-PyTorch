# Google-Quickdraw-Classification-with-PyTorch
Simple PyTorch neural network that makes predictions for Google Quickdraw images, though I only used 4 random ones, those being:
* Violin (index 0)
* Zebra (index 1)
* Trumpet (index 2)
* Hamburger (index 3)


Funny thing, I actually originally had guitar instead of hamburger, but because violins and guitars look so similar, it was greatly reducing the score of my model. When I replaced guitar with hamburger, the score went from 77% to 91%.

I put a lot of documentation in the notebook itself, so check that out if you're interested for some reason
